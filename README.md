# Grafos_Tableau
Una recopilacion de los grafos más usados al representar datos con la herramienta Tableau. Este repositorio recoge varias formas de representación de una fuente de datos ejemplo(tablas, grafos, diagramas...) con comentarios y explicaciones en cada uno de ellos.

Para poder visualizar el contenido has de descargar **Curso_Tableau.twb** y abrirlo con la aplicación de Tableau.
